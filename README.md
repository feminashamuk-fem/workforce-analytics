Workforce Analytics Dashboard

Project Overview

This project analyzes employee attrition patterns using HR data to understand workforce behavior and identify key factors influencing employee turnover.

Tools Used

Excel (Data Cleaning)

SQL (Data Analysis)

Power BI (Dashboard & Visualization)

Objectives

Identify factors affecting employee attrition

Analyze high-risk job roles and age groups

Study impact of salary, overtime, and satisfaction

Build interactive HR dashboard

Key Metrics

Total Employees

Employees Left

Attrition Rate

Average Salary

Average Tenure

Key Insights

Overtime employees show higher attrition

Younger employees (25–34) leave more frequently

Low salary groups have higher turnover

Certain job roles are more affected by attrition
